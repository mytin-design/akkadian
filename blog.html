HTML (HyperText Markup Language).

HTML stands for HyperText MarkUp Language. It is the default markup language that defines the layout of all websites. To achieve this, it has elements also known as tags. Examples of tags include, html, head, title, header, main, section, article, footer, e.t.c. Syntax of a web page normal flow: html The elements are divided into segments, and must be used accordingly to achieve a website normal flow. Elements have tags, that add additional features. Increasing the usability of the same in speficied instances. Exampes of attributes are name, content, href, rel, target, max, e.t.c. Elements in HTML are divided into categories. These categories are landmark elements, sectioning elements etc. Examples of landmark elements include html, head, nav, main and footer. The landmark elements define the basic layout, and defin the main sections of a web page. Sectioning elements divide landmark elements into small sections. Examples include: section, article , etc. Since HTML is the skeleton of the web page. It can be linked to other styling languages such as CSS or Bootstrap to make sense of most HTML parts.

CSS (Cascading Style Sheet).

Cascading style sheets (CSS) is a styling language. To effect styles, the sheet is linked to the language that requires styling. To achieve this, css uses declarations. A declaration is made using an element name, class or id defined in an element in HTML. The syntax: 

                        
                        elementName {
                          property: value;
                          property: value;
                          (as many properties as needed.)
                        }
                        .class {
                          property: value;
                          property: value;
                          (as many properties as needed.)
                        }
                        #id {
                          property: value;
                          property: value;
                          (as many properties as needed.)
                        }
In the above css code syntax, an element name, class or id has been used to declare a set of styles. The declared styles are enclosed within curly braces. A declaration has a property and a value. For example: 
                        
                          elementName {
                            color: blue;
                          }
A property is separated from the value using a colon. One may declare as my prperties as possible and as needed.

Js (JavaScript)

JavaScript is a scripting programming language. It is mostly used for web development and design. It was developed by Brendan Eich in June 1995. Javascript language makes websites dynamic. Dynamic websites retrieve, access, and update data on the website from the server periodically. Javascript syntax encompasses a variety of features that enable it fulfill its functions. Examples include: 

                        
                          keyword identifier = 'value';
                          const, var, let, functions, methods, 
objects, and classes.
var, const, let define and store variables. A variable is a piece of data, used for various functions in js code. Example: 
                        
                          const varName = 'My variable';
                          var varName = 'My variable';
                          let varName = "My variable";
arrays are storage locations for related data or data sets. syntax: 
                        keyword identifier = ['array1, array2, array3, arrayn'];
                      
                        
                          
                          const array = ['one', 'two', 'three', 'four', 'five'];
functions, group related code, and reusable code. syntax: 
                        
                          function functionName() {
                            //statements
                          }

                          functionName();
Example: 
                        
                          function learnCode() { // function keyword and function name
                            console.log("Learn code"); //code 
                          }
                          learnCode(); //A function call;
A function's syntax is designed to enclose related data. Starts with a function keyword, function name, code statements enclosed in curly braces and a function call;
Objects An object stores large amounts of related code. syntax: let objectName = { name: 'value', name: 'value' } Example: 
                        
                          let codeLessons = {
                            'lessonOne': 'HTML',
                            'lessonTwo': 'CSS',
                            'lessonThree': 'Js'
                          }
An object is made up of a keyword, object name, and key-pair values enclosed in curly braces.
